Guess The Word Game

What is this project about?

It’s a fun interactive word game that uses JavaScript and JSON files. They have a certain number of guesses to figure out the word before before the game ends. You can repeat the game as many times as you like.

Why did you want to build this project?

This was the final project for the JavaScript class at Skillcrush. I was excited to build it because it gave me a chance to apply everything I learned in the class. It was a great check to see if I had actually retained any of the information I learned. Plus it is a cool project that anyone can see the value in, not just other coders.

What hurdles/problems did you run into?

I definitely got stuck on a couple of parts. I struggled to get the bullets to show up correctly on the screen as placeholders. I also had a bug that was causing the number of guesses to change once you clicked the button to play the game again.

How did you overcome these problems?

Sometimes it can be overwhelming to think of all the things you need to figure out to make something work. In most cases if I just break down the problem into tiny pieces, my brain has a better chance of figuring out the problem. In this case I took each issue separately and went through the code to see how this part was broken and what I needed to do to fix it.

What did you learn from this project?

I learned that even the things that look simple are not always so simple! I also learned the value of breaking problems down into smaller pieces. It’s also helpful for me to write down each of the steps I need to complete to make something work so that I can complete them one by one without getting overwhelmed by the big picture.
