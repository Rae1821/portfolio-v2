GitHub Repo Gallery

What is this project about?

The GitHub repo gallery project uses the GitHub API to fetch the data from the repos on my GitHub account. The public repos from my account are then displayed on the page, allowing the user to get information about each repo, search for a particular repo, or view the repo on GitHub itself.

Why did you want to build this project?

This was a project for the JavaScript fundamentals class at Skillcrush. I was excited to build this project because it was a chance to learn how APIs work. Being about to bring data into your site or application dynamically, instead of hard coding it, really changes the game in terms of what you are able to build.

What hurdles/problems did you run into?

I had some issues displaying the name of the repos on the page at first. Although I was fetching the information it was not displaying on the page. I also decided to go in and change the default colors to something that matched my color scheme.

How did you overcome these problems?

At this point I have truly realized the value in breaking down problems into smaller pieces. I went back through the steps I needed to do to show the repos on the page. I realized that I was trying to display the repos without looping over them so I added in a for of loop and was able to fix this problem.

To change up the colors I used the developer tools in Chrome to find the classes I needed to target in my CSS. I’m not sure what I would do without the developer tools! I use them all of the time.

What did you learn from this project?

I learned the value of fetching information from APIs and much of a game changer that is. The project gave me great practice using the knowledge I learned in my JavaScript class. I learned anything is possible as long as you don’t give up!
